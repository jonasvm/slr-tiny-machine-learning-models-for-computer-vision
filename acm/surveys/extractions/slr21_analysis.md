## 2. Scope and Purpose of the Review

**Main objective of the review:**  
To survey machine learning systems deployed at the network edge, including techniques for training and inference on resource-constrained devices, as well as the software frameworks and hardware platforms that enable edge intelligence.

**Problem the review claims to solve:**  
Deploying machine learning models directly on IoT and edge devices is challenging due to limited computational resources, storage constraints, latency issues associated with cloud offloading, increased communication costs, and privacy concerns.

**Scope:**  
Edge AI, machine learning at the network edge, resource-constrained devices, IoT systems, distributed ML systems, edge computing architectures.

**Main contributions claimed by the authors:**

- Survey of classical machine learning and deep learning techniques used in edge computing environments.
- Review of hardware platforms and software frameworks supporting edge AI.
- Analysis of applications and operational aspects of ML systems deployed at the network edge.
